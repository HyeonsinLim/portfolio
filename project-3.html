시스템·자동화 구축2026.05 · 1개월

HR 인건비 리포트 개발

예산 관리 툴이 없던 환경에서 BigQuery와 스프레드시트로 실시간 인력·인건비 리포트를 직접 설계해 경영진 의사결정을 상시 지원했습니다.

배경

설립 6년차임에도 보안상 이유로 당근마켓과 동일한 어드민 프로그램을 사용할 수 없어 예산 관리 툴 자체가 부재했습니다. ERP 개발자 채용도 4개월 이상 소요되는 상황이었고, 전표 처리는 당근마켓 재무실이 대행하다 보니 예산 사용 내역을 상세히 조회할 권한조차 없어 근거 데이터 없이 인건비를 관리해야 했습니다.

프로젝트 프로세스

01
문제
  • 예산 관리 툴 부재, ERP 개발자 채용 장기화
  • 예산 사용 내역 조회 권한 없어 관리 근거 데이터 자체가 없음
02
접근
  • 보안팀에 BigQuery 접근 권한 요청 및 확보
  • 구성원 raw 데이터 기반 구글 스프레드시트 리포트 설계
03
실행
  • 인력현황(입퇴사자, 퇴사사유 비율, 평균 근속기간) 구조화
  • 인건비현황(성과급·연장근무 비율, 직급별 연봉밴드), 연간 예산 대비 소진율 반영
  • 정규직 채용 시 계약직 조정 규모 시뮬레이션 기능 구현
  • 인력현황 매일 자정, 인건비 매월 급여마감 시점 자동 갱신 설계
04
결과
  • 요청마다 raw 데이터 재가공하던 구조 → 실시간 상시 확인 구조로 전환
  • 경영진의 인력 운영 의사결정을 데이터 기반으로 상시 지원
