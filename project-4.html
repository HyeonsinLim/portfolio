CJ그룹 채용 시스템 'TAS' 개편

흩어져 있던 채용 데이터를 통합 대시보드로 구축해 의사결정 속도를 높이고, 지원자 만족도와 평가 효율을 동시에 개선했습니다.

배경

CJ그룹 임원·경력직 채용 과정의 진행 현황을 실시간으로 파악할 방법이 없어, 채용 단계별 데이터가 흩어진 채로 관리되고 있었고 의사결정에 필요한 현황 파악에 시간이 걸리는 구조였습니다.

프로젝트 프로세스

01
문제
  • 채용 단계별 데이터가 흩어져 실시간 파악 불가
  • 현황 파악에 시간이 걸려 의사결정 지연
02
접근
  • 이해관계자(인사담당자·면접관) 요구사항 수집 및 분석
  • 채용 전 과정 통합 모니터링 대시보드 데이터 모델링
03
실행
  • 개발자와 대시보드 화면 UX 설계·검증
  • 런칭 후 사용자 교육 및 가이드 문서 작성
04
결과
  • 채용 전 과정 실시간 조회로 의사결정 속도 단축
  • 지원자 만족도 '채용 투명성' 항목 15%p 향상
  • 평가자 역량 분석 리포트 자동화로 연 200시간 이상 효율화
  • 채용데이터·이직률·성과지표 연계 경영진용 대시보드 초기버전 확장
